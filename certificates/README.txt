CERTIFICATE IMAGES FOLDER
========================

Place your certificate images in this folder with the following names:

1. python-cert.jpg  - Python Programming Certificate
2. java-cert.jpg    - Java Development Certificate
3. nodejs-cert.jpg  - Node.js Backend Certificate
4. mongodb-cert.jpg - MongoDB Developer Certificate

INSTRUCTIONS:
-------------
1. Save your certificate images as JPG or PNG format
2. Recommended size: 1200x900 pixels (or similar aspect ratio)
3. Rename them according to the names above
4. The portfolio will automatically display them

If certificate images are not available:
- The icon will be shown instead as a fallback
- Everything else will work normally

The portfolio is designed to handle missing images gracefully.
